**Clustering & Classification Approach**

## Deskripsi Proyek

Proyek ini bertujuan untuk **mendeteksi potensi kecurangan (fraud)** pada transaksi perbankan menggunakan pendekatan **machine learning**, yaitu **clustering (unsupervised learning)** dan **klasifikasi (supervised learning)**.

Dataset yang digunakan adalah **Bank Transaction Dataset for Fraud Detection** yang berisi **2.512 sampel transaksi**, mencakup:

* atribut transaksi,
* demografi pelanggan,
* serta pola penggunaan rekening.

Dataset ini cocok untuk **analisis anomali dan fraud detection** pada sistem keuangan.

## Tujuan Proyek

1. Mengidentifikasi **pola transaksi normal dan tidak normal** menggunakan clustering
2. Membangun model klasifikasi untuk **memprediksi transaksi fraud**
3. Membandingkan performa beberapa algoritma machine learning
4. Memberikan insight terhadap karakteristik transaksi mencurigakan
